# 2025-22k1122：売上予測モデル検証用データセット

公開データ（Amazon Sales RankおよびGoogle Trends）を用いた売上予測モデル構築のためのデータセットとプログラムを公開しています。

## 本リポジトリについて
国内で入手が困難な実売上データの代わりに、入手が容易なランキングデータと検索トレンドを活用し、データ制約下でどの程度の予測精度が得られるかを検証することを目的としています。

## 文献情報
津田 悠太：公開データを活用した疑似的な購買指標に基づく売上予測モデル

## ファイル構成
解析に使用したデータおよびプログラムは以下の通りです。用途に応じてご活用ください。

### データセット
- **camera/**：デジタルカメラの製品特徴、ランキング、検索推移データ及びそれらの取得スクリプト
- **washing/**：洗濯機の製品特徴、ランキング、検索推移データ及びそれらの取得スクリプト

### プログラム
- **ASIN_2023-2025.py**：2023-2025期間のASINデータ日次抽出スクリプト
- **ASIN_週次_2023-2025.py**：2023-2025期間のASINデータ週次抽出スクリプト
